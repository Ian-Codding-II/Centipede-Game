# Centipede Game

Classic arcade Centipede game built with SFML 2.5 in C++.

## Features
- Classic Centipede gameplay
- Multiple difficulty settings
- Player customization
- Leaderboard system

## Building

See [DEVELOPMENT.md](DEVELOPMENT.md) for setup instructions.
```bash
make
make run
```
